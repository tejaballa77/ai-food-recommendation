## 🥗 AI Disease-Based Food Recommendation System

This project is an AI-powered application that provides **personalized food recommendations** based on a user’s health condition and budget constraints. It processes natural language input (e.g., *“I have diabetes suggest food under 300”*) and delivers clear suggestions on what to eat and what to avoid.

The system follows a modular workflow that includes **NLP processing, disease-based rule extraction, menu analysis, and a recommendation engine**, ensuring fast and accurate results. It is designed to help users make **health-conscious food decisions in real time**.

The application is built using **Python and Streamlit**, deployed on the cloud for public access, and optimized to deliver responses within seconds.

---

### 🚀 Key Highlights

* 🧠 Understands natural language queries
* 🩺 Disease-specific food recommendations
* 💰 Budget-aware filtering
* ⚡ Fast response (2–3 seconds)
* 🌐 Deployed and accessible via web
* 📱 QR-based quick access

---

### 🔄 Workflow

User Input → NLP Processing → Disease Identification → Diet Rule Extraction → Menu Analysis → Recommendation Engine → Response Generation → UI Display

This project demonstrates how AI can be applied to solve real-world problems in **healthcare, nutrition, and intelligent recommendation systems**.
